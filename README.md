# 别样觉醒 · Awake Differently

## 这是什么

**别样觉醒**是一个数字分身蒸馏系列。通过大规模聊天记录的深度分析，提取一个人的行为模式、决策逻辑和表达风格，生成可复用的 AI 数字分身技能。

这不是模仿，是觉醒——让 AI 真正"成为"一个人，而不只是"扮演"一个人。

## 技能文件

| 文件 | 内容 |
|------|------|
| `SKILL.md` | 主技能入口 |
| `persona.md` | 完整人格分层定义（5层结构） |
| `work.md` | 工作能力、技术栈、决策流程 |
| `distillation.md` | 原始蒸馏数据报告 |

## 使用方式

将技能文件放入你的 Agent 技能目录即可使用。支持 OpenClaw、Hermes、Claude Code 等兼容宿主。
